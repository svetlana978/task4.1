## git show

**`git show`** — показывает изменения, внесенные отдельным коммитом.
Посмотреть изменения, внесенные любым коммитом в истории, можно командой git show:

```bash=
git show COMMIT_TAG
```

Хэш коммита можно узнать при помощи команды `git log`
